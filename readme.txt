Para ejecutar el programa principal tienes que poner en la terminal de Visual Studio Code o bien en cualquier otra terminal
pero desde la carpeta subgrup-prop41.5 el siguiente comando: 

                            - make run-Presentation-Windows (WINDOWS)
                            - make run-Presentation-LinMac (LINUX Y MAC)


En la carpeta EXE tienes los ejecutables creados.

En la carpeta FONTS tienes el codigo de todo el trabajo separado en carpetas por capas.

Cabe destacar que si te sale un error relacionado con los perfiles la primera vez que ejecutes tienes que borrar los JSON que aparecen encima de la carpeta DOCS.
